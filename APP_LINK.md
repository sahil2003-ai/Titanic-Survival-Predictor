## About the Application

This is a live, interactive machine learning web application built with Streamlit. The app predicts the survival probability of Titanic passengers based on their characteristics.

### Features

- **Real-time Predictions**: Input passenger details and get instant survival probability
- **Interactive Interface**: User-friendly input form with sliders and dropdowns
- **Trained Model**: Uses a logistic regression model trained on the Titanic dataset
- **Instant Results**: Immediate prediction with probability score

### How to Use

1. Visit the live application link above
2. Enter the passenger's details in the form:
   - Passenger Class (1, 2, or 3)
   - Sex (Male/Female)
   - Age (in years)
   - Number of Siblings/Spouses aboard
   - Number of Parents/Children aboard
   - Fare (ticket price)
3. Click the "Predict Survival" button
4. View the prediction result and probability

### Technology Stack

- **Framework**: Streamlit
- **Machine Learning**: scikit-learn (Logistic Regression)
- **Data Processing**: pandas, numpy
- **Hosting**: Streamlit Cloud

### Note

This is a demonstration application designed for educational purposes. The predictions are based on historical data patterns and should not be taken as definitive survival indicators.
